8、P14  经营情况讨论与分析
(1)2019营收121.4亿，较2018年的145.5亿下滑16.6%。
主要原因：受宏观经济影响，2019年中国广告市场需求疲软，公司客户结构中互联网内类广告主因市场融资环境等原因调减广告预算，互联网行业广告收入大幅缩减构成了报告期内公司营收下滑的主要原因。
曙光:传统行业客户的收入持续增长，下半年营收下滑幅度收窄
(2)公司自2018年起大幅扩张电梯媒体自营，导致2019年主营业务成本较2018年上涨35.3%
(3)报告期捏受宏观经济影响，客户回款速度普遍放缓，导致账龄结构恶化，风险增加，股公司的信用减值损失的计提和拔备也相应增加了85%
(4)综上原因，2019年全年实现营业利润23.7亿，同比下滑66%
(5)2018年7月，阿里及其关联方战略入股分众。阿里赋能数字化分众
(6)截止2019年末，公司通过集中竞价方式共回购2.43亿股，购买最高价8.64元/股，最低价5.04元/股，实际累计支付15.3亿，计入库存货15.3亿。

(一)会计政策变更的情况说明
2019.1.1起执行新金融工具准则。主要变更内容如下:
①所有已确认金融资产其后续均按摊余成本或公允价值计量

③将自然资产减值会计处理由"已发生损失法"修改为"预期损失法"，要求考虑金融资产未来预期信用损失情况，从而更加及时、足额地计提金融资产减值准备
