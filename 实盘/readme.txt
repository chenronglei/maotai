1、港股通当日购买的股票，在【成交查询】可以查到 挂盘价(港币报价)、成交量、成交的人民币总额(不包含交易费用) 
       注:上面查询结果与【港股通】-【成交查询】里查询的是一致的，【港股通】有更详细的结算汇率(该汇率与实时汇率基本一致)
  
   但在【资金流水】查不到当日购买的股票流水，得过2个交易日后才显示，且日期显示的是2个交易日的日期。

  比如20211015以20.9港元购买了600股的小米：
      在【成交查询】里查到的是20211015号，以20.9港元购买了600股的小米，成交金额10371.21元(0.827*12540)
      在【资金流水】里查到的是20211019号，以20.9港元购买了600股的小米, 发生金额10394.36元(0.827*12540+14.06印花税+6.22手续费+1.65结算费)

总结：想查看港股通真实成交价格，可以在成交后2个工作日之后查询【资金流水】的发生金额(包含交易过程中的各种费用，是真实的费用，汇率以购买日收盘时的汇率计算)


2、购买的成本与持仓成本为什么差距较大？  主要是折算汇率低于实际汇率，导致以实际产生的人民币成交金额除以低汇率，得到的成本(港币)比购买时的要高
比如20211015以20.9港元购买了600股的小米：
      在【成交查询】里查到的是20211015号，以20.9港元购买了600股的小米，成交金额10371.21元(0.827*12540)
      在【资金流水】里查到的是20211019号，以20.9港元购买了600股的小米, 发生金额10394.36元(0.8268*12540+14.06印花税+6.22手续费+1.65结算费)

在20211104号查看股票，显示折算汇率为0.797，成本(港币)21.736,这个成本的计算如下(随着这是汇率变动，这个成本(港币)还上会变动的，但实际以人民币计算成本是不变的)：
10394.36/0.797/600=21.736

在20211104号查看股票，持仓市值为9779.19，现价(港币)20.45.持仓市值计算如下：
600*20.45*0.797=9779.19

盈亏 -615.17  盈亏比 -5.92%，计算如下
9779.19-10394.36 = -615.17
-615.17/10394.36 = -5.92%

产生亏损比实际的要大：主要是折算汇率低于实际汇率，导致折算成人民币的持仓市值偏低
